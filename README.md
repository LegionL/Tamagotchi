# Tamagotchi


### Download 

Open your Terminal and run:

`git clone git@github.com:LegionL/Tamagotchi.git`

### Install Requirements

Open your terminal and run:

`pip install -r requirements.txt`

### Run Tests

Open your terminal and run:

`py test.py -b`

### Run Game

Open your terminal and run:

`py Tamagotchi.py`

Good Luck! Have Fun!
